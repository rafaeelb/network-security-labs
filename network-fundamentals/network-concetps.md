IP -> Endereço lógico que é atribuído a cada dispositivo da rede.<br>
MAC -> Endereço físico na placa de rede, que não pode ser alterado. Serve como identificação para se conectar ao Switch<br>
DNS -> Domain Name System, traduz endereços IP em nomes como por exemplo www.google.com<br>
HTTP -> protocolo de requisição que opera na porta 80<br>
HTTPS -> protocolo de requisição com segurança que opera na porta 443<br>
