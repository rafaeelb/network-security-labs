IP -> Endereço lógico que é atribuído a cada dispositivo da rede.
MAC -> Endereço físico na placa de rede, que não pode ser alterado. Serve como identificação para se conectar ao Switch
DNS -> Domain Name System, traduz endereços IP em nomes como por exemplo www.google.com
HTTP -> protocolo de requisição que opera na porta 80
HTTPS -> protocolo de requisição com segurança que opera na porta 443
