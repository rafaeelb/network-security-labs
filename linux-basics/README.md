Comandos linux

mkdir->criar repositório(pasta)  
cd->entrar em uma pasta  
cd .. ->volta uma pasta  
cd~ -> vai pra home  
ls->listar diretórios e arquivos  
ls -l ->lista detalhada: exemplo: -rw-r--r-- 1 rafael rafael 25 arquivo.txt("-"=arquivo, "rw-"=dono pode ler/escrever, "r--"=grupo só lê,
"r--"=outros só leem  
pwd->mostra onde você está no sistema (diretório)  
touch->criar um arquivo vazio  
echo->escrever e ">"=jogar dentro do arquivo  
">" = sobrescreve o arquivo  
">>" = adiciona sem apagar  
cat->mostra o conteúdo do arquivo  
cp origem destino->copia um arquivo  
mv antigo novo->move ou renomeia  
rm-> remove o arquivo  
chmod-> altera as permissões de um arquivo ou pasta  
grep->procura uma palavra dentro de um arquivo (conteúdo dentro dos arquivos)  
whoami->mostra o usuário atual  
id-> mostra seu usuario, seu grupo e grupos que voce pertence  
find-> procura arquivos no sistema (somente arquivos)  

valores numericos dos grupos  
R = 4  
W = 2  
X = 1
